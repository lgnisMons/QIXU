## 1. SEO 策略

### 1.1 Landing Pages

每个具有 SEO 价值的服务+地域组合需要独立 Landing Page：

- 独立 URL
- 独立 Meta Title / Description
- 独立 Schema.org 结构化数据
- 独立内容（不使用模板化重复内容）

初始 Landing Page 列表：

- 深圳家教
- 深圳AI家教
- 深圳小学家教
- 深圳初中家教
- 深圳高中家教
- 深圳英语家教
- 深圳高考志愿规划
- 深圳AI课程
- AI训练营
- AI学习助手

### 1.2 技术 SEO

- 使用 Next.js App Router 服务端渲染
- 所有页面提供 Meta 标签
- 结构化数据（JSON-LD）
- Sitemap 自动生成
- robots.txt 配置
- 图片优化（next/image）
- Core Web Vitals 达标

### 1.3 内容 SEO

- 文章系统：AI 教程、学习方法、教育资讯
- FAQ 系统
- 导师/课程页面的独特内容
- 内链策略

## 2. GEO（Generative Engine Optimization）

针对 AI 搜索引擎（ChatGPT、Claude、Perplexity 等）优化：

- 结构化内容便于 AI 抓取引用
- 权威信息来源标注
- FAQ 格式内容
- 清晰的实体定义（关于启序、服务、导师等）

## 3. 内容运营

### 3.1 内容类型

- **AI 教程**：AI 工具使用教程、Claude Code 教程等
- **学习方法**：学习方法论、时间管理、考试技巧
- **免费资料**：可下载的学习资源
- **工具导航**：AI 工具推荐与评测
- **FAQ**：常见问题解答

### 3.2 内容原则

- 有价值、可操作
- 不制造焦虑
- 不标题党
- AI 辅助但人工审核

## 6. 数据隐私

### 6.1 游客数据

- 默认不保留
- AI 分析结果可下载
- 关闭页面自动清除临时数据

### 6.2 用户数据

- 用户决定数据保存范围
- 支持数据导出
- 支持数据删除
- 支持重新授权

### 6.3 AI 数据

- 明确告知 AI 使用范围
- AI 分析结果标注"AI生成"
- AI 不主动推销

## 8. 转化路径

核心转化目标不是购买课程，而是：

> 让访客进入 Growth Sequence，建立成长档案，成为长期学习者。

转化路径：

```
浏览 → 体验AI → 下载报告 → 注册成长档案 → 课程/训练营 → 作品 → 成长 → 长期用户
```
