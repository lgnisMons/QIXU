# Command: ui-add

Add a shadcn/ui component.

```bash
pnpm --filter web ui:add [component-name]
```

Examples:
```bash
pnpm --filter web ui:add button
pnpm --filter web ui:add card
pnpm --filter web ui:add input
pnpm --filter web ui:add dialog
```

Components are added to `apps/web/src/components/ui/`.
