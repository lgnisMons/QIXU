# QIXU — AI Education Platform

A specification-first, capability-based monorepo for building an AI-native education platform.

## Repository Map

| Directory | Purpose |
|-----------|---------|
| `apps/` | Deployable applications (Next.js App Router) |
| `packages/` | Shared libraries, UI kit, utilities |
| `specs/` | Feature specifications — one per capability |
| `decisions/` | Architecture Decision Records (ADRs) |
| `docs/` | Permanent project documentation |
| `scripts/` | Build, CI, and utility scripts |
| `.github/` | CI/CD workflows |
| `.vscode/` | Shared editor settings |

## Getting Started

Read `docs/` in order, then pick a spec from `specs/`.
